はじめに Introduction
ようこそ Welcome
WACCA RATING ANALYZER は、WACCAのマイページで確認できる情報を元に、レーティングを上昇させる上で便利なデータを確認できるプログラムです。 The WACCA RATING ANALYZER provides you useful data for increasing your rating based on the data available on your WACCA My Page.
特徴 Features
データ収集なし No data collection
このプログラムはブラウザー上で動作し、あなたのプレイ成績がサーバーへ送信されることはありません。 This program runs in your browser and does not send your play results to the server.
レーティング候補譜面の上限数なし Show more 'Rating Candidate Charts'
レーティング候補譜面をはじめ、レーティングを上げるために便利なデータをすべての譜面で見ることができます。 You can see 'Rating Candidate Charts' and other useful data for increasing your rating on all charts.
レーティング上昇量が詳細に分かる Detailed rating increase
所定のスコアへ到達した際のレーティングの上昇量が、譜面ごとに分かります。 You can see the amount of increase in your rating when you reach a certain score for each chart.
注意 Caution
ブラウザーがプライベートモードである場合、タブを閉じると分析されたデータは失われます。なお再度プログラムを実行することで、同じ結果を得られます。 If your browser is in private mode, closing a tab will cause the analyzed data to be lost. But don't worry. Just run the program again and you will get the same result.
譜面定数はあくまで推測された値であるため、公式の値とは異なる可能性があります。 The chart constants are only guessed values and may be different from the official values.
このプログラムは暇つぶしを目的に作成されているため、品質は保証されておりません。役に立たなかったらごめんなさい。 The quality of this program is not guaranteed, as it was created as a way to pass the time. I'm sorry if this program did not help you...
準備 Getting Started
1. このページをブックマークする 1. Bookmark this page.
ブックマークの名前の変更は任意です｡ *Renaming your bookmarks is optional.
環境によっては、当プログラムは動作しない場合があります。まずお使いのブラウザーで動作確認を行い、動作しない場合はFirefoxに切り替えてみてください。Chromeでは動作しなかったという報告がいくつかあります。 This program may not work on some devices. So please try to see if this program works in your current browser, and if not, switch to Firefox. I have received reports that this program did not work on Chrome.
ブックマークの名前の変更は任意ですが、Android端末をご利用の場合は、当プログラムを実行する際にブックマークの名前を入力する必要がありますので、入力が容易な名前に変更してください。 Renaming your bookmarks is optional, but if your device is an Android device, you will need to enter the name of the bookmark when you run this program, we recommends that you change the name to one that is easy to enter.
2. ブックマークレットをコピーする 2. Copy the bookmarklet.
ブックマークレットとは、任意のタイミングでブックマークを開くことで実行できるJavaScriptで書かれたスクリプトのことです。ここで保存したブックマークレットをマイページで実行すると、ブラウザーがあなたのプレイ成績を収集します。 A bookmarklet is a script written in JavaScript that you can run by opening a bookmark at any time. If you run the bookmarklet you saved here on My Page, the browser will collect your play scores.
3. ステップ1で作成したブックマークのアドレスを、ステップ2でコピーした文字列に書き換えて保存する 3. Rewrite the address of the bookmark created in step 1 to the string copied in step 2 and save it.

定数テーブル Constants Table
キーワード keywords
曲名の一部を指定できます。
You can set a part of the song title.
let は、曲名に「let」を含む曲を検索します。 let will search for songs that contain "let" in the song title.
big bang は、曲名に「big」および「bang」を含む曲を検索します。 big bang will search for songs that contain "big" and "bang" in the song title.
level:
レベルの値を指定できます。 You can set the value of the level.
level:12 は、レベル「12」の譜面を含む曲を検索します。 level:12 will search for songs that contain a chart of level "12".
const:
譜面定数の値を指定できます。 You can set the value of the chart constants.
const:13.4 は、譜面定数「13.4」の譜面を含む曲を検索します。 const:13.4 will search for songs that contain a chart of constant "13.4".
newer:
新曲枠であるか否かを検索します。trueまたはfalseを指定できます。 Searches for "newer charts", can be true or false.
newer:true は、新曲枠である曲を検索します。 newer:true will search for songs that are in the newer charts.
複数のオプションを組み合わせることもできます。 You can also combine multiple options.
level:14 stronger は、レベル「14」かつ曲名に「stronger」を含む曲を検索します。 level:14 stronger will search for songs that have a level of "14" and contain "stronger" in the song title.
全文検索にも対応しています｡ Full-text search is also supported.
例えば、 14.1（「const:」オプションなし） は、いずれかの項目に「14.1」を含む曲を検索します。つまり譜面定数「14.1」の譜面を含む曲を検索するのと ほぼ 同じです。 For example, 14.1 (without "const:" option) will search for songs that contain "14.1" in one of the fields. That means it is almost the same as searching for songs that contain the chart constant "14.1".
